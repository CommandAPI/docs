# docs

Documentation for CommandAPI.

## How to build

### Build website

```bash
yarn install
```

```bash
yarn docs:build
```

### View website changes locally

```bash
yarn docs:dev
```

### Verify reference code

```bash
cd reference-code
./gradlew build
```
